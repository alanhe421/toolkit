# 工具集

将常用web工具实现

## 已实现功能

+ 编码解码
  + base64编码
  + MD5加密
  + 格式化json数据
  + URL编码
  + Unix时间戳转换
  + UTF-8转码
+ 颜色板
+ 加密解密
  - AES
+ 用户端信息
    - 获取本机IP
    - 获取本机浏览器信息

## 下步实现功能
+ WHOIS
+ 在线生成特定尺寸图片
+ 正则表达式检测

+ 测试本机网速
+ 在线反编译APK,支持下载
+ 图表，支持下载，包括文字云常用图表
+ 在线执行JS脚本
